# Attention Design Router

Use one main attention mechanic and at most one secondary mechanic.

## Attention Mechanics

| Mechanic | Use when | Prompt direction |
|---|---|---|
| `oversized_headline` | most covers, hot takes, offers, list promises | huge readable type, 2-8 words, high contrast |
| `human_gaze` | creator, beauty, fashion, personal story, video cover | face close-up, direct gaze, expressive reaction |
| `macro_texture` | beauty, food, material, gadgets, craft | extreme close-up, tactile texture, gloss/steam/fiber detail |
| `hard_flash` | street, fashion, youth, nightlife, raw creator energy | direct flash, sharp shadow, intentional snapshot feel |
| `color_block` | discounts, list covers, dopamine style, strong category hooks | saturated blocks, clear separation, bold palette |
| `surreal_scale` | launch, premium object, tech, gift, concept cover | giant object, tiny scene, exaggerated but clean |
| `problem_scene` | pain-point content, before state, mistakes, conflict | messy/awkward scene, visible tension, before-result contrast |
| `creator_proof` | TikTok, Reels, Kuaishou, Shorts, practical demos | hands, selfie angle, unboxing, real-use trace |
| `handwritten_markup` | Xiaohongshu, Pinterest, Bilibili, guide covers | arrows, circles, notes, sketch labels, useful annotations |
| `collage_layer` | fashion, gift, trend, event, resources | cutouts, stickers, layered props, editorial rhythm |
| `diagonal_motion` | sports, travel, food, cleaning, action | tilted composition, blur, motion line, dynamic crop |
| `cinematic_light` | premium, electronics, fragrance, serious topic | dramatic shadow, rim light, atmospheric glow |
| `saveable_checklist` | XHS/Pinterest guides, Bilibili explainers, community posts | clean card, large promise, useful mini-list |
| `event_culture_hook` | festival, sports, fandom, seasonal trend | one cultural anchor, not symbol clutter |

## Event/Fandom Bias

For sports, fandom, holidays, and global events, route to lifestyle scenes instead of official-event posters:

- watch party, home party, cafe, travel setup, fan outfit, beauty prep, food table, big-screen setup, creator commentary.
- Use inspired palettes and original props.
- Avoid forcing the user/product into every image. Often the best cover is a product-absent scene that makes the occasion feel urgent.
