Richpost Theme Guide

Richpost themes package layout tokens, page masters, and static assets for reusable long-form visual posts.

Bundle Structure

  • theme.json: theme metadata, slots, and supported surfaces.
  • layout.tokens.json: spacing, typography, color, and export metrics.
  • masters/*.master.html: cover, body, ending, or custom section masters.
  • assets/*: local images, SVGs, and other static theme resources.

Minimal Theme

{
  "id": "crossroads-mint",
  "name": "Crossroads Mint",
  "version": "1.0.0",
  "surfaces": ["richpost"],
  "masters": {
    "cover": "masters/cover.master.html",
    "body": "masters/body.master.html",
    "ending": "masters/ending.master.html"
  }
}

Implementation Notes

Keep templates deterministic and offline-friendly. Theme bundles should not execute arbitrary JavaScript or depend on remote assets at render time.
