# Roadmap Archives

## Purpose

本目录用于保留已关闭事项的执行证据，覆盖优化主线任务与历史文档完成索引，避免主文档只看起来“干净”却丢失历史。

## 命名规范

- 文件名：
  - `completed-tasks-archive-YYYY-MM-DD.md`
  - `completed-docs-archive-YYYY-MM-DD.md`
- 建议频率：每周或每次里程碑后归档
- 每个文件必须记录：
  - 执行日期
  - 完成项标识（任务 ID 或文档名）
  - 完成说明
  - 验证结果
  - cleanup
  - 残留风险

## 归档流程

1. 任务状态改为 `done`。
2. 补齐 `evidence`、`cleanup`、`next_step`。
3. 在本目录新增或追加快照文件。
4. 主文档里保留 `RM` 入口和最小索引，不保留完整细节。
