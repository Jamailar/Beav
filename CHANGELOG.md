# RedBox 更新日志

## v2.1.0 (2026-05-20)

### 数字人视频
- 新增数字人视频生成链路，支持使用角色参考视频和配音音频生成口播视频
- 角色库支持保存参考视频，数字人创作可直接复用角色的视频、声音和基础设定
- 数字人生成接入媒体任务系统，支持提交、等待、解析成片地址和回到工作台查看结果
- 数字人入口并入生成工作台，减少角色、音频、视频结果之间来回切换
- 支持基于角色视频进行音色克隆，让数字人口播更接近角色本身的声音表现

### 音频与音色
- 音频生成支持按语言筛选音色，降低跨语言选错音色导致的生成质量问题
- 音色列表跟随 TTS 模型变化，不同模型只展示对应可用音色
- 修复模型绑定的声音克隆槽位，角色的多模型音色配置更稳定
- 修复 TTS 模型列表来源，避免页面选项和真实可调用能力不一致

### 知识库与采集
- 浏览器插件新增知乎内容采集，知乎页面内容可以进入 RedBox 知识库并参与后续检索
- 修复知乎采集路由和索引路径，提升采集后入库与检索稳定性
- 优化知识库视频转写重试，减少临时上游失败导致的视频资料入库失败
- 知识库空状态增加插件入口提示，首次配置外部采集更直接

### 媒体与聊天
- 聊天输入框支持直接粘贴图片，并作为附件进入当前对话和 Agent 工具调用
- 媒体结果增加保存相关操作，生成后的图片、音频、视频更容易沉淀复用
- 支持更多 Windows 本地媒体路径，减少本地媒体素材预览或生成时的路径访问失败

### 拖拽文件快捷功能
- 支持将本地文件直接拖入聊天窗口，快速作为附件加入当前对话
- 拖入图片、视频等媒体文件后，会自动进入后续预览、分析和 Agent 工具调用链路
- 浏览器插件支持拖拽网页图片保存到 RedBox 素材库，减少从网页素材到本地素材库的手动下载步骤

### 修复
- 修复聊天快捷提示词被显示到正式消息里的问题
- 修复已有 RedClaw 空间无法进入新样式流程的问题
- 修复官方 AI 设置中的缓存用量显示问题

## v2.0.3 (2026-05-16)

### 媒体与知识库
- 聊天内视频附件改为内嵌播放器，自动保持视频比例，并支持封面帧预览
- 优化知识库媒体导入流程，支持更稳定地处理本地媒体资源路径
- 修复知识库目录分页查询，避免分页结果不完整

### 桌面体验
- 改进桌面端页面响应速度，减少聊天、稿件、生成工作台等页面切换和刷新时的卡顿
- Windows 登录页支持透明无边框标题栏控制，和主窗口标题栏表现保持一致

### 打包与分发
- 同步桌面端版本号到 2.0.3
- 重新生成 macOS、Windows 与浏览器插件发布资产

## v1.11.0 (2026-04-29)

### RedClaw 内联文件预览
- 新增 RedClaw 对话内文件预览功能，支持侧边分屏显示
- 预览面板自动平衡分屏布局，始终保持在输入框上方
- 支持 post / markdown 类型条目的内联预览
- 文件预览路径解析更加稳定可靠

### RedClaw 聊天体验
- 新增聊天成员 @ 提及功能，支持在输入框中 @ 召唤团队成员
- 输入框占位提示改为「使用 # 调用知识库 / 使用 @ 召唤团队成员」
- 附件上传状态实时显示
- 修复首次进入 RedClaw 时引导提示闪烁问题
- 修复 RedClaw 图片生成任务的进度订阅稳定性

### 可视化知识索引（全新）
- 将图片 OCR 索引系统全面升级为视觉语义索引（visual manifests）
- 知识库搜索结果中直接展示视觉匹配内容
- 新增视觉块详情和审计元数据
- 自动发现并索引可视化知识文件
- 支持索引增量持久化，进度实时可见
- 可视化索引增加诊断信息与候选预览
- 索引内容在知识页面中可全文搜索
- 删除知识源时自动清理对应索引产物
- 可视化索引配置并入 AI 模型设置页面

### 文件索引面板
- 新增文件索引状态面板，展示索引进度
- 面板数据支持缓存以提升设置页性能
- 索引看板随进度事件实时刷新
- Advisor 知识变更后自动重建索引

### 封面生成
- 修复封面生成任务在页面切换后消失的问题

### 知识库
- 修复 Windows 下知识条目 ID 与路径兼容性问题

### 稿件与创作者工具
- 改进稿件 AI 差异审阅体验
- 加速小红书博主笔记侧边栏采集速度
- 主体库工作区数据持久化

### 认证与系统
- 改进聊天运行时的认证与错误处理
- 新增 AI 认证失败时的登录恢复引导
- 修复官方配置同步时自定义 AI 路由被覆盖的问题

### 更新与分发
- 应用更新渠道切换至 RedBox 下载源
