# RedBox 更新日志

## v1.11.0 (2026-04-29)

### RedClaw 内联文件预览
- 新增 RedClaw 对话内文件预览功能，支持侧边分屏显示
- 预览面板自动平衡分屏布局，始终保持在输入框上方
- 支持 redpost / markdown 类型条目的内联预览
- 文件预览路径解析更加稳定可靠

### RedClaw 聊天体验
- 新增聊天成员 @ 提及功能，支持在输入框中 @ 召唤团队成员
- 输入框占位提示改为「使用 # 调用知识库 / 使用 @ 召唤团队成员」
- 附件上传状态实时显示
- 修复首次进入 RedClaw 时引导提示闪烁问题
- 修复 RedClaw 图片生成任务的进度订阅稳定性

### 可视化知识索引（全新）
- 将图片 OCR 索引系统全面升级为视觉语义索引（visual manifests）
- 知识库搜索结果中直接展示视觉匹配内容
- 新增视觉块详情和审计元数据
- 自动发现并索引可视化知识文件
- 支持索引增量持久化，进度实时可见
- 可视化索引增加诊断信息与候选预览
- 索引内容在知识页面中可全文搜索
- 删除知识源时自动清理对应索引产物
- 可视化索引配置并入 AI 模型设置页面

### 文件索引面板
- 新增文件索引状态面板，展示索引进度
- 面板数据支持缓存以提升设置页性能
- 索引看板随进度事件实时刷新
- Advisor 知识变更后自动重建索引

### 封面生成
- 修复封面生成任务在页面切换后消失的问题

### 知识库
- 修复 Windows 下知识条目 ID 与路径兼容性问题

### 稿件与创作者工具
- 改进稿件 AI 差异审阅体验
- 加速小红书博主笔记侧边栏采集速度
- 主体库工作区数据持久化

### 认证与系统
- 改进聊天运行时的认证与错误处理
- 新增 AI 认证失败时的登录恢复引导
- 修复官方配置同步时自定义 AI 路由被覆盖的问题

### 更新与分发
- 应用更新渠道切换至 RedBox 下载源
- 新增发布版本下载量统计脚本
- 版本号更新，构建 macOS 与 Windows 安装包
